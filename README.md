# AP-AMS
<!-- prettier-ignore-start -->
<!-- markdownlint-disable-next-line MD036 -->
_✨ 一个简单的适用于拓竹的自动换色系统 ✨_
<!-- prettier-ignore-end -->
</div>

## 特点
- 一个换色单元就可以自己工作！无需上位机！
- 成本低，单个换色单元仅花费**15元**
- 接入homeassistant，数据可视化，方便监视换色以及纠错(待完善)
- 使用esp8266作为MCU,Arduino开发
- 换色单元主体除电机和基础齿轮均为3d打印
- 使用舵机控制齿轮松紧，仅换料时压紧齿轮，正常打印依靠拓竹的近端挤出机
- 正常打印低噪音
